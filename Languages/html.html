HTML Cheatsheet By DevOasis
  1. HTML Boilerplate: 
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    // Write your code here //
</body>
</html>
  2. Div Section : 
    <div> Block element </div>
  3. Heading Tags : 
    <h1>Page title</h1>
<h2>Subheading</h2>
<h3>Sub subheading</h3>
<h4>Quaternary heading</h4>
  4. Paragraph with style : 
    <p style="text-align: center;">text</p>
  5. Image : 
    <img src="/demo.jpg" alt="description" height="48" width="100">
  6. Text Links : 
    <a href="https://html6.com/" target="_blank" rel="nofollow">
    Click here
</a>
  7. Bold text : 
    <strong>Bold text</strong>
  8. Italic text : 
    <em>Italic text</em>
  9. underline text : 
    <span style="text-decoration: underline;">Underlined text</span>
  10. Iframe Tags : 
    <iframe src="link.html" width="200" height="200"></iframe>
  11. Comments in HTML : 
    <!-- HTML Comment -->
  12. Horizontal Line : 
    <hr/>
  13. Line Break : 
    <br/>
  14. Tables in HTML : 
    <table><caption>Phone numbers</caption>

    <th colspan="2">Phone</th>
    </tr>

    </tr>
    <tr>

    <tr>
    <td> </td>

</tfoot>
</table>
  15. Unordered list : 
    <ul>
<li>First</li>
<li>Second</li>
<li>Third</li>
</ul>
  16. Ordered list : 
    <ol>
<li>First</li>
<li>Second</li>
<li>Third</li>
</ol>
  17. Form in HTML : 
    <form action="/action.php" method="post">
